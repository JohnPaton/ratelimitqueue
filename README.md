# RateLimitQueue

A thread safe, rate limited Python queue.


## Installation

To install `ratelimitqueue`, clone the repository and install with `pip`:

```bash
$ git clone https://github.com/JohnPaton/ratelimitqueue.git
$ cd ratelimitqueue
$ pip install .
```
